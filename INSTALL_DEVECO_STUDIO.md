# ⚠️ 重要更新

DevEco Studio 现已不再支持 Linux 系统！

请在 **Windows 10/11** 系统下编译鸿蒙版。

## 📝 详细指南

请查看：**HARMONYOS_WINDOWS_COMPILE_GUIDE.md**

## 🚀 Windows 编译步骤

1. 在 Windows 上下载 DevEco Studio
   - 访问：https://developer.harmonyos.com/cn/develop/deveco-studio
   - 下载 Windows 版本

2. 安装 DevEco Studio
   - 运行安装程序
   - 安装 HarmonyOS SDK API 12

3. 打开项目
   - File -> Open -> F:\o2ochat\harmony\O2OChat

4. 编译项目
   - Build -> Build Hap(s) / APP(s)
   - 输出：entry\build\outputs\entry-default-signed.hap
