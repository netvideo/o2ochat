## 🚀 使用方法

### 桌面版 (立即可用)

**Linux:**
```bash
./o2ochat-linux --help
./o2ochat-linux
```

**macOS:**
```bash
./o2ochat-macos --help
./o2ochat-macos
```

**Windows:**
```cmd
o2ochat-windows.exe --help
o2ochat-windows.exe
```

### Android (需 Java 后)

```bash
sudo apt install openjdk-17-jdk
export ANDROID_HOME=$HOME/android-sdk
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
cd android
./gradlew assembleDebug
```
