## 简介

### 什么是 O2OChat？

O2OChat 是一款**纯 P2P 即时通讯软件**，无需中央服务器，所有通信直接在用户之间进行。

### 核心特性

- 🔒 **端到端加密** - AES-256-GCM 加密
- 🌐 **纯 P2P 架构** - 无需中央服务器
- 📱 **多平台支持** - Windows、macOS、Linux、Android、iOS、HarmonyOS
- 🌍 **多语言支持** - 16 种语言
- 🚀 **高性能** - 传输速度 >100 MB/s

### 技术优势

| 特性 | O2OChat | 传统 IM |
|------|---------|---------|
| **服务器依赖** | 无 | 需要 |
| **隐私保护** | 端到端加密 | 服务器可访问 |
| **数据控制** | 用户完全控制 | 服务商控制 |
| **运行成本** | 零 | 高 |

## 安装指南

### Windows

#### 下载

1. 访问 GitHub Releases
2. 下载 `o2ochat-windows.exe`
3. 保存到桌面或程序文件夹

#### 安装

```
双击 o2ochat-windows.exe
程序自动启动
```

#### 首次运行

```
1. 允许防火墙访问（如提示）
2. 程序自动创建 Peer ID
3. 进入主界面
```

### macOS

#### 下载

1. 访问 GitHub Releases
2. 下载 `o2ochat-macos`
3. 移动到 Applications 文件夹

#### 运行

```
1. 右键点击 -> 打开
2. 首次运行需确认
3. 程序启动
```

### Linux

#### 下载

```bash
wget https://github.com/netvideo/o2ochat/releases/download/v4.0.0/o2ochat-linux
chmod +x o2ochat-linux
```

#### 运行

```bash
./o2ochat-linux
```

### Android

#### 安装 APK

```
1. 下载 app-debug.apk
2. 允许"未知来源"安装
3. 点击安装
4. 完成
```

#### 从应用商店

```
1. 打开 Google Play / 应用市场
2. 搜索 "O2OChat"
3. 点击安装
```

## 快速开始

### 第一步：创建账号

```
1. 启动应用
2. 自动创建 Peer ID
3. 记录 Peer ID（用于添加好友）
```

**示例 Peer ID**:
```
QmXyZ123456789abcdef
```

### 第二步：添加联系人

#### 方法 1: 扫描 QR 码

```
1. 点击"+"添加联系人
2. 选择"扫描二维码"
3. 对准对方 QR 码
4. 确认添加
```

#### 方法 2: 手动输入

```
1. 点击"+"添加联系人
2. 选择"手动输入"
3. 输入对方 Peer ID
4. 输入昵称
5. 确认添加
```

### 第三步：开始聊天

```
1. 点击联系人
2. 输入消息
3. 点击发送
```

## 故障排除

### 无法连接

#### 症状

```
- 显示"连接中..."
- 消息发送失败
- 联系人显示离线
```

#### 解决方法

```
1. 检查网络连接
2. 重启应用
3. 检查防火墙设置
4. 尝试切换网络（WiFi/移动数据）
```

### 消息发送失败

#### 症状

```
- 消息显示红色感叹号
- 长时间显示"发送中"
```

#### 解决方法

```
1. 检查网络连接
2. 重新发送
3. 重启应用
4. 检查对方是否在线
```

### 通话质量问题

#### 症状

```
- 通话卡顿
- 声音断续
- 视频模糊
```

#### 解决方法

```
1. 检查网络速度（建议>1Mbps）
2. 关闭其他占用带宽的应用
3. 切换到 WiFi 网络
4. 降低视频质量（设置中）
```

### 文件传输失败

#### 症状

```
- 传输中断
- 速度极慢
- 文件损坏
```

#### 解决方法

```
1. 检查剩余存储空间
2. 检查网络连接
3. 重新发送
4. 尝试分段发送大文件
```

### 应用崩溃

#### 症状

```
- 应用突然关闭
- 无法启动
- 界面卡死
```

#### 解决方法

```
1. 强制关闭应用
2. 清除缓存
3. 重启设备
4. 重新安装应用
```

## 常见问题

### Q1: O2OChat 安全吗？

**A**: 非常安全！

- ✅ 端到端加密（AES-256-GCM）
- ✅ 无中央服务器存储
- ✅ 消息仅存在于设备
- ✅ 开源代码可审计

### Q2: 需要服务器吗？

**A**: 不需要！

O2OChat 是纯 P2P 架构，所有通信直接在用户设备间进行。

### Q3: 如何备份聊天记录？

**A**: 

```
设置 -> 存储 -> 备份聊天记录
选择备份位置 -> 开始备份
```

### Q4: 可以同时在多个设备使用吗？

**A**: 可以！

每个设备会生成不同的 Peer ID，但可以通过联系人列表同步功能保持联系。

### Q5: 消息会保存多久？

**A**: 

- 本地消息：永久保存（除非手动删除）
- 云端：无云端存储
- 对方设备：由对方决定

### Q6: 如何删除消息？

**A**:

```
长按消息 -> 选择"删除"
选择"仅删除"或"删除并通知对方"
```

### Q7: 支持群聊吗？

**A**: 支持！

- 创建群组
- 邀请成员
- 群组管理
- 群消息加密

### Q8: 文件大小有限制吗？

**A**: 无限制！

但建议大文件使用分段发送。

### Q9: 支持视频通话吗？

**A**: 支持！

- 语音通话
- 视频通话
- 屏幕共享（部分平台）

### Q10: 如何举报违规用户？

**A**:

```
用户资料 -> 右上角"..." -> 举报
选择举报原因 -> 提交
```
