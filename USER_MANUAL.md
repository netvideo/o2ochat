## 简介

### 什么是 O2OChat？

O2OChat 是一款**纯 P2P 即时通讯软件**，无需中央服务器，所有通信直接在用户之间进行。

### 核心特性

- 🔒 **端到端加密** - AES-256-GCM 加密
- 🌐 **纯 P2P 架构** - 无需中央服务器
- 💻 **多平台支持** - Windows、macOS、Linux、Android
- 🌍 **多语言支持** - 16 种语言
- 🚀 **高性能** - 传输速度 >100 MB/s

### 界面类型

O2OChat 提供两种界面：

1. **命令行界面** - 所有平台通用（当前版本）
2. **图形界面** - Android APK

## 安装指南

### Windows

#### 下载

1. 访问 GitHub Releases
2. 下载 `o2ochat-windows.exe`
3. 保存到桌面或程序文件夹

#### 运行

```
双击 o2ochat-windows.exe
程序自动启动
```

### macOS

#### 下载

1. 访问 GitHub Releases
2. 下载 `o2ochat-macos`
3. 移动到 Applications 文件夹

#### 运行

```
./o2ochat-macos
```

### Linux

#### 下载

```bash
wget https://github.com/netvideo/o2ochat/releases/download/v4.0.0/o2ochat-linux
chmod +x o2ochat-linux
```

#### 运行

```bash
./o2ochat-linux
```

### Android

#### 安装 APK

```
1. 下载 app-debug.apk
2. 允许"未知来源"安装
3. 点击安装
4. 完成
```

## 设置与配置

### 配置文件

创建 `config.json` 文件：

```json
{
  "nickname": "我的昵称",
  "dataDir": "./data",
  "logLevel": "info"
}
```

### 命令行参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `--config` | 配置文件路径 | `--config config.json` |
| `--data-dir` | 数据目录 | `--data-dir ./mydata` |
| `--help` | 显示帮助 | `--help` |
| `--version` | 显示版本 | `--version` |

## 故障排除

### 无法启动

#### 症状

```
程序闪退
或
权限被拒绝
```

#### 解决方法

```bash
# Linux/macOS
chmod +x o2ochat
./o2ochat

# Windows
右键 -> 以管理员身份运行
```

### 无法连接

#### 症状

```
显示"连接中..."
消息发送失败
```

#### 解决方法

```
1. 检查网络连接
2. 检查防火墙设置
3. 重启程序
```

### 文件传输失败

#### 症状

```
传输中断
速度极慢
```

#### 解决方法

```
1. 检查剩余存储空间
2. 检查网络连接
3. 重新发送
```

## 常见问题

### Q1: O2OChat 安全吗？

**A**: 非常安全！

- ✅ 端到端加密（AES-256-GCM）
- ✅ 无中央服务器存储
- ✅ 消息仅存在于设备
- ✅ 开源代码可审计

### Q2: 需要服务器吗？

**A**: 不需要！

O2OChat 是纯 P2P 架构，所有通信直接在用户设备间进行。

### Q3: 如何备份聊天记录？

**A**: 

数据存储在 `./data` 目录，备份该目录即可。

### Q4: 可以同时在多个设备使用吗？

**A**: 可以！

每个设备会生成不同的 Peer ID。

### Q5: 消息会保存多久？

**A**: 

- 本地消息：永久保存（除非手动删除）
- 云端：无云端存储

### Q6: 如何删除消息？

**A**:

删除数据目录中的对应文件。

### Q7: 支持群聊吗？

**A**: 开发中！

### Q8: 文件大小有限制吗？

**A**: 无限制！

### Q9: 支持视频通话吗？

**A**: 开发中！

### Q10: 如何举报违规用户？

**A**:

拉黑对方 Peer ID 即可。
