## برامج المراسلة الفورية P2P النقية

O2OChat هو برنامج مراسلة فورية نظير إلى نظير (P2P) نقي لا يعتمد على خوادم مركزية لتخزين الرسائل. تتم جميع الاتصالات مباشرة بين المستخدمين.

### الميزات الأساسية

- 🔒 **التشفير من النهاية إلى النهاية** - تستخدم جميع الرسائل تشفير AES-256-GCM
- 🌐 **بنية P2P النقية** - لا يوجد خادم مركزي، اتصال مباشر
- 📱 **دعم متعدد المنصات** - Android و iOS و Windows و Linux و macOS و HarmonyOS
- 📁 **نقل الملفات** - استئناف النقل المتقطع، التنزيل متعدد المصادر، التحقق من شجرة Merkle
- 🌍 **16 لغة** - الصينية والإنجليزية واليابانية والكورية والألمانية والفرنسية والإسبانية والروسية والماليزية والعبرية والعربية والتبتية والمنغولية والأويغورية والصينية التقليدية

### البدء السريع

```bash
# استنساخ المشروع
git clone https://github.com/yourusername/o2ochat.git
cd o2ochat

# البناء
go build -o o2ochat ./cmd/o2ochat

# التشغيل
./o2ochat
```

### هيكل المشروع

```
o2ochat/
├── cmd/              # نقاط الدخول
├── pkg/              # المكتبات الأساسية
│   ├── identity/     # إدارة الهوية
│   ├── transport/    # النقل الشبكي
│   ├── signaling/    # خدمة الإشارة
│   ├── crypto/       # وحدة التشفير
│   ├── storage/      # تخزين البيانات
│   ├── filetransfer/ # نقل الملفات
│   └── media/        # معالجة الصوت/الفيديو
├── ui/               # واجهة المستخدم
├── cli/              # أدوات سطر الأوامر
├── tests/            # الاختبارات
├── docs/             # الوثائق
└── scripts/          # سكريبتات البناء
```

### حزمة التقنيات

- **Go 1.21+** - النواة الخلفية
- **Protocol Buffers** - التسلسل
- **QUIC/WebRTC** - نقل P2P
- **SQLite** - التخزين المحلي
- **Fyne** - واجهة المستخدم الرسومية للكمبيوتر
- **Jetpack Compose** - واجهة مستخدم Android
- **SwiftUI** - واجهة مستخدم iOS
- **ArkTS** - واجهة مستخدم HarmonyOS
