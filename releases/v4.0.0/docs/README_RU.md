## Пиратская программа P2P обмена сообщениями

O2OChat — это программа обмена сообщениями в реальном времени по сети P2P (peer-to-peer), которая не зависит от центральных серверов для хранения сообщений. Все сообщения отправляются прямо между пользователями.

### Основные функции

- 🔒 **Шифрование от конца к концу** — все сообщения используют шифрование AES-256-GCM
- 🌐 **Архитектура чистой P2P** — нет центрального сервера, прямая связь
- 📱 **Поддержка нескольких платформ** — Android, iOS, Windows, Linux, macOS, HarmonyOS
- 📁 **Передача файлов** — возобновление прерванной передачи, многопоточная загрузка, проверка по дереву Меркла
- 🌍 **16 языков** — китайский, английский, японский, корейский, немецкий, французский, испанский, русский, малайский, иврит, арабский, тибетский, монгольский, уйгурский, традиционный китайский

### Быстрый старт

```bash
# Клонировать проект
git clone https://github.com/yourusername/o2ochat.git
cd o2ochat

# Собрать
go build -o o2ochat ./cmd/o2ochat

# Запустить
./o2ochat
```

### Структура проекта

```
o2ochat/
├── cmd/              # Точки входа
├── pkg/              # Основные библиотеки
│   ├── identity/     # Управление идентификацией
│   ├── transport/    # Сетевой транспорт
│   ├── signaling/    # Служба сигнализации
│   ├── crypto/       # Модуль шифрования
│   ├── storage/      # Хранение данных
│   ├── filetransfer/ # Передача файлов
│   └── media/        # Обработка аудио/видео
├── ui/               # Пользовательский интерфейс
├── cli/              # Инструменты командной строки
├── tests/            # Тесты
├── docs/             # Документация
└── scripts/          # Скрипты сборки
```

### Стек технологий

- **Go 1.21+** — ядро бэкенда
- **Protocol Buffers** — сериализация
- **QUIC/WebRTC** — P2P-транспорт
- **SQLite** — локальное хранилище
- **Fyne** — настольный GUI
- **Jetpack Compose** — Android UI
- **SwiftUI** — iOS UI
- **ArkTS** — HarmonyOS UI
