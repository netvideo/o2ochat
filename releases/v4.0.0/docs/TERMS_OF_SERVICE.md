## 6. P2P 通信风险

### 6.1 技术风险

使用 P2P 通信，您应了解以下风险：

| 风险 | 说明 | 缓解措施 |
|------|------|---------|
| **IP 地址可见** | 对方可能看到您的 IP | 使用 VPN 或 Tor |
| **元数据泄露** | 通信时间、频率可能暴露 | 减少使用频率 |
| **网络攻击** | 可能遭受 DDoS 等攻击 | 使用防火墙 |
| **设备安全** | 设备被黑可能泄露数据 | 保持系统更新 |

### 6.2 安全建议

我们建议：
1. 使用 VPN 隐藏真实 IP
2. 定期更新应用版本
3. 不随意添加陌生人为联系人
4. 不点击可疑链接
5. 不接收来源不明的文件

### 6.3 加密限制

虽然我们使用强加密，但请注意：
- ⚠️ 加密不能保证 100% 安全
- ⚠️ 量子计算机可能威胁现有加密
- ⚠️ 社会工程学攻击无法通过加密防御
- ⚠️ 设备被黑会绕过所有加密
