# 集成测试套件

## 测试目标

验证各模块间的接口兼容性和协同工作能力。

## 测试范围

### 模块间集成测试

1. **身份 + 信令** - 消息签名验证集成
2. **身份 + 加密** - 密钥管理集成
3. **信令 + 传输** - 连接建立集成
4. **传输 + 文件传输** - 文件传输集成
5. **加密 + 存储** - 加密存储集成
6. **媒体 + 传输** - 音视频通话集成

### 端到端测试

1. **用户注册流程** - 身份创建 → 存储 → 信令注册
2. **P2P 连接流程** - 信令交换 → 传输建立 → 数据通信
3. **文件传输流程** - 分块 → Merkle 树 → 多源下载 → 验证
4. **音视频通话流程** - 设备初始化 → 编解码 → RTP 传输 → 通话管理

## 测试环境

### 必需条件
- Go 1.18+
- 网络连接（用于 WebSocket 测试）
- 临时存储空间

### 测试配置
```bash
# 运行所有集成测试
go test ./tests/integration/... -v

# 运行特定测试
go test ./tests/integration/... -run TestIdentitySignaling

# 运行测试并显示覆盖率
go test ./tests/integration/... -cover
```
